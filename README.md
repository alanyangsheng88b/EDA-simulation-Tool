# EDA Simulation Tool

一个纯浏览器运行的 EDA 电路仿真工作台，提供示例电路、可拖拽元器件、参数面板、SPICE 风格网表和瞬态波形展示。

## 功能

- 拖拽或点击添加电压源、电阻、电容、电感、二极管、接地和探针。
- 在参数面板中调整幅值、频率、电阻、电容、电感和二极管导通压降。
- 自动生成 SPICE 风格网表，并支持复制。
- 运行瞬态分析，展示输入/输出曲线、输出峰值和 RMS 等测量值。
- 内置桥式整流、RC 一阶充电、RLC 串联谐振三个示例电路。

## 本地运行

```bash
npm run start
```

然后访问 <http://127.0.0.1:4173/>。

## 代码检查

```bash
npm run check
```
